Provide methods for common file and registry operations. Include localjs_file.js to use it.

iniReadInt ( String  filename,
String  section,
String  key,

Read an integer from INI file.

Parameters:
[in] filename INI file name to read from
[in] section Section in INI file
[in] key Key of the value to read from
[in] default_value default value to be returned if the section and key do not exist.
Returns:
The integer read from INI file, or default_value if the section and key do not exist.

Code below read an integer from INI file; if not found, return -1: 

    var intVal = LOCALJS.FILE.iniReadInt("c:\\settings.ini", "section_name", "key_name", -1);

iniWrite ( String  filename,
String  section,
String  key,
var  value 
)

Write an entry to INI file.

Parameters:
[in] filename INI file name to write to
[in] section Section in INI file
[in] key Key of the value to write
[in] value String value to write to. vlaue will be converted to string if it's not. If value cannot be converted to a string (e.g., pass in undefined), the key will be deleted from INI file.

Code below saves settings to INI file: 

    LOCALJS.FILE.iniWrite("c:\\settings.ini", "section_name", "key_name", "value");

To remove an entry from ini file:

    LOCALJS.FILE.iniWrite("c:\\settings.ini", "section_name", "key_name", undefined);
