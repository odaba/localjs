Creates a new browser window or standalone Internet Explorer and navigates to given url.

Parameters:
[in] url The url the new browser window or standalone Internet Explorer navigates to.
[in] left Optional. The initial horizontal position of the window. Refer to parameter x of CreateWindow Function. If omitted, the default value is to center the window on the screen.
[in] top Optional. The initial vertical position of the window. Refer to parameter y of CreateWindow Function. If omitted, the default value is to center the window on the screen.
[in] width Optional. The width, in device units, of the window. Refer to parameter nWidth of CreateWindow Function. If omitted, the default value is 300.
[in] height Optional. The height, in device units, of the window. Refer to parameter nHeight of CreateWindow Function. If omitted, the default value is 180.
[in] window_style Optional. The style of the window being created, ignored by standalone Internet Explorer. Refer to parameter dwStyle of CreateWindow Function. LOCALJS.UI predefines some predefined window style:

  • LOCALJS.UI.WS_CLOSE_ONLY: The browser window has only a close button on the title bar.
  • LOCALJS.UI.WS_NO_TITLE_BAR: The browser window doesn't have a title bar.
  • LOCALJS.UI.WS_NORMAL: The browser window has a normal title bar. This is the default value if window_style is omitted.
[in] parent_window Optional. The HWND handle to the parent or owner window of the browser window being created, ignored by standalone Internet Explorer. To use the HWND handle of the current browser window, pass in localJS.hostWnd. If omitted, the default value is zero. Refer to parameter hWndParent of CreateWindow Function.
[in] initFunction Optional. A JavaScript function which will be executed when the new browser window or standalone Internet Explorer loads page. The JavaScript function accepts one parameter: the JavaScript window object of the browser window page.
[in] leave_alone Optional. If set to false, the new browser window or standalone Internet Explorer will be closed when current browser window reloads page or is closed. It set to true, the new browser window or standalone Internet Explorer is not closed when current browser window reloads page or is closed. The default value is false if omitted.
[in] ie Optional. If set to true, a standalone Internet Explorer will be created. If set to false, a LocalJS browser window will be created. The default value is false if omitted.
Returns:
The BrowserWindow object for the new created browser window or standalone Internet Explorer.

To create a new browser window without title bar: 

    var browser_window = LOCALJS.UI.newWindow("http://myapp.com/", 20, 30, 400, 300, LOCALJS.UI.WS_NO_TITLE_BAR);

To inject a JavaScript function to the new created browser window: 

    var funcInjected = function(new_jswindow)
    {
        // access the DOM node of the new browser window page:
        var new_html = new_jswindow.document.getElementsByTagName('html')[0];
    };

    var browser_window = LOCALJS.UI.newWindow("http://myapp.com/", 20, 30, 400, 300, LOCALJS.UI.WS_NO_TITLE_BAR, localJS.hostWnd, funcInjected);
Attention:
If parent_window is localJS.hostWnd and ie is false or omitted, the new created browser window will always be closed when current browser window is closed.
