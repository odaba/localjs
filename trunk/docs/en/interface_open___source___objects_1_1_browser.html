Detailed Description

Open source object. Represent a browser window. Returned by LOCALJS.UI.createBrowser. Include localjs_ui.js to use it.

Method Documentation

window getJSWindow (  ) 

Returns the JavaScript window object of the browser page. Returns null before the page is fully loaded in browser.

You can use window object to attach JavaScript function in current page to the event in the Browser page:

    var browser = LOCALJS.UI.createBrowser(0, "http://example.com/exampleApp/", 100, 60, 400, 250, LOCALJS.UI.WS_NORMAL, localJS.hostWnd);
    for (;;)
    {
        var browser_window = browser.getJSWindow();
        if (null == browser_window)
            continue; // the window object may not be available before the page in the browser fully loaded.
        if (!LOCALJS.UI.doEvents()) // doEvents returns false means page is being unloaded
            break;
        if (browser_window)
            browser_window.attachEvent("onunload", function() { alert("ByeBye"); });
        break;
    }

var getJSVariable ( String  variable_name  ) 

Returns the global variable or function of the browser page by name. Returns null before the page is fully loaded in browser.

Parameters:
[in] variable_name The name of the global variable or function defined in Browser page.

You can access a global function or variable defined in the Browser page from current page:

    var browser = LOCALJS.UI.createBrowser(0, "http://example.com/exampleApp/", 100, 60, 400, 250, LOCALJS.UI.WS_NORMAL, localJS.hostWnd);
    for (;;)
    {
        var browser_function = browser.getJSVariable("global_function_name");
        if (null == browser_function)
            continue; // the variable may not be available before the page in the browser fully loaded.
        if (!LOCALJS.UI.doEvents()) // doEvents returns false means page is being unloaded
            break;
        if (browser_function)
            browser_function("I got you!");
        break;
    }

boolean isRunning (  ) 

Returns true if the Browser window is not closed, false if it has been closed.

If the Browser window has been closed by user or by Browser.close method, isRunning method returns false.

Note:
To get a notification when the Browser window is closed, hook the onunload event of the window object of the browser page.

Close the browser window.
