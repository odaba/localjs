Class List

Here are the classes, structs, unions and interfaces with brief descriptions:
BrowserWindowOpen source object. Represents a browser window. Returned by LOCALJS.UI.newWindow. Include localjs_ui.js to use it
BufferBuffer object represents a fixed size buffer, used as parameters in Dll function call
COMObject COM exposes methods to create COM object, connect COM events, etc
DllCallDllCall object enables JavaScript call functions exported by DLL or EXE files, including most Windows API functions
EnumeratorSimilar to Enumerator object in JScript, providing access to items in Collection object
EventHookReturned by method COM.connectEvents, used to disconnect COM event connected
FILEOpen source object. Provide methods for common file and registry operations. Include localjs_file.js, localjs_ui.js and localjs_webservice.js to use it
localJSThe root object of all advanced JavaScript objects. Available directly in JavaScript.
LOCALJSThe root object of all open source JavaScript objects. Need to include JavaScript files from LOCALJS Open Source Library to be available
StructureStructure object represents a C struct, used as parameters in Dll function call
ThreadRepresent a script thread
ThreadingThreading object exposes methods to create new script thread, sleep, get thread id, etc
WindowHookHook messages of the specified window
