Detailed Description

Open source object. Provides method for web service call. Include localjs_webservice.js to use it.

Method Documentation

callUrl ( String  http_method,
String  url,
[optional] Object  callback,
[optional] String  request_body,
[optional] String  username,
[optional] String  password 
)

Call web service asynchronously.

Parameters:
[in] http_method String that specifies the HTTP method used to open the connection: such as "GET", "POST", or "HEAD". This parameter is not case-sensitive. Refer to parameter sMethod of open Method of XMLHttpRequest Object.
[in] url String that specifies either the absolute or a relative URL of the Web service. Refer to parameter sUrl of open Method of XMLHttpRequest Object.
[in] callback Optional. Callback functions. Details are discussed below.
[in] request_body Optional. Variable that specifies the body of the message being sent with the request. It may be a String, array of unsigned bytes, or an XML Document Object Model (DOM) object. Refer to parameter varBody of send Method of XMLHttpRequest Object.
[in] username Optional. String that specifies the name of the user for authentication. If this parameter is empty string ("") or missing and the site requires authentication, a logon window is displayed. Refer to parameter sUser of open Method of XMLHttpRequest Object.
[in] password Optional. String that specifies the password for authentication. This parameter is ignored if the username parameter is empty string ("") or missing. Refer to parameter sPassword of open Method of XMLHttpRequest Object.

A typical callback parameter is defined as below: 

    var callback = {'ok' : function(contentText, oHttp) { /* read ok handling */ },
                    'fail' : function(statusCode, oHttp) { /* error handling */ } };

If the call succeeds, callback.ok is called with the response text returned by web service (parameter contentText). If the call fails, callback.fail is called with the HTTP status code returned by web service (parameter statusCode). In both cases, oHttp is the XMLHttpRequest Object used for the web service call.

Following code illustrates how to use callUrl method to call web service:

    var onOK = function (responseText, objHttp)
    {
        // assume the service returns JSON format string
        var json_response = (new Function("return " + responseText))();
    };

    var onFail = function (statusCode, objHttp)
    {
        alert("Web service call failed with " + statusCode);
    };

    LOCALJS.WEB_SERVICE.callUrl("GET", "http://localjs.org/demo/demo_ws.php", { 'ok' : onOK, 'fail' : onFail });
