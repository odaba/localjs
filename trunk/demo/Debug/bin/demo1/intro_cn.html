WPF 的华丽，Native C++ 的轻巧      只需要  HTML+CSS+JavaScript
LocalJS 演示程序文件列表
 LocalJS Library 点击展开
  • Debug \ bin \ LocalJS.dll：LocalJS 的 DLL 文件。
  • Debug \ bin \ localJS.cfg：localJS 的配置文件，必须和 LocalJS.DLL在同一目录下。
 LocalJS JavaScript Open Source Library 点击展开
  • Debug \ bin \ demo1 \ localjs \ localjs_file.js：实现 namespace LOCALJS.FILE，提供了常见的文件操作接口。
  • Debug \ bin \ demo1 \ localjs \ localjs_ui.js：实现 namespace LOCALJS.UI，提供了常见的界面功能接口。
  • Debug \ bin \ demo1 \ localjs \ localjs_webservice.js：实现 namespace LOCALJS.WEB_SERVICE，提供了 web service 调用接口。
 在注册表和 INI 文件中保存设置 点击展开
  • Debug \ bin \ demo1 \ _js \ global_functions.js：在用户个人目录下创建子目录，用于保存 INI 文件。
  • Debug \ bin \ demo1 \ _js \ program_settings.js：在注册表中读、写、删除键值。读写 INI 文件中的整数值。
  • Debug \ bin \ demo1 \ _js \ personal_settings.js：读写 INI 文件中的字符串。
 截取来自 COM 事件和窗口函数的退出通知，让用户有机会确认退出 点击展开
  • Debug \ bin \ demo1 \ _js \ exit_confirm.js：LOCALJS.UI 把复杂的 COM 事件处理和窗口函数消息截取包装成一个简单的回调函数：LOCALJS.UI.exitCallback。
 创建新窗口及多个窗口之间的互操作 点击展开
  • Debug \ bin \ demo1 \ _js \ demo_newwindow.js：在 DHTML 事件处理函数中用 LOCALJS.UI.createBrowser 函数创建新窗口，通过 initFunction 参数，将主窗口中定义的一个 JavaScript 函数注入到新窗口中执行，从而实现窗口之间的互操作。
  • Debug \ bin \ demo1 \ new_window_cn.html：demo_newwindow.js所打开的页面。
  • Debug \ bin \ demo1 \ _js \ demo_google.js：当用户选择在新窗口中打开链接的时候，可以通过回调函数 LOCALJS.UI.newWindowCallback 来决定是否取消这次点击；或者创建一个新窗口，在此新窗口中打开此链接。
 利用 ADO 对象访问数据库 点击展开
  • Debug \ bin \ demo1 \ _js \ demo_ado.js：打开数据库演示窗口。如果数据库演示窗口已经打开，则切换到该窗口。
  • Debug \ bin \ demo1 \ data_table_cn.html：数据库演示窗口的页面文件。
  • Debug \ bin \ demo1 \ _js \ demo_datatable.js：用 ADO 对象打开数据库并在汉化的 Yahoo YUI Library 的 DataTable 控件中排序、编辑。
  • Debug \ bin \ demo1 \ sample data.xls：作为数据库的 Excel 文件。
  • Debug \ bin \ demo1 \ yui：Yahoo YUI Library 的一部分。
 调用任意 domain 的 web service 点击展开
  • Debug \ bin \ demo1 \ _js \ demo_webservice.js：通过 LOCALJS.WEB_SERVICE.callUrl 调用示例 web service，利用 JavaScript 的特点非常简单地将 JSON 格式的字符串转变为 JavaScript 中的变量。
  • demo_ws.php：被调用的示例 web service 的源程序。
 调用 Windows API 并和用户创建的 DLL 中的函数互相调用 点击展开
  • Debug \ bin \ demo1 \ _js \ demo_structure.js：调用 LOCALJS.UI 中的函数移动窗口和打开文件选择对话框。
  • Debug \ bin \ demo1 \ _js \ demo_calljs.js：调用演示程序附带的 LocalJSDemoDll.dll 所输出的函数 callJSFunctionInNewThread；callJSFunctionInNewThread 再调用 JavaScript 中的函数。
  • LocalJSDemoDll \ LocalJSDemoDll.dsp：LocalJSDemoDll 的 Visual C++ 6 项目文件。
 创建 native script 线程，主线程和新线程之间互相调用 点击展开
  • Debug \ bin \ demo1 \ _js \ demo_thread.js：在新线程中执行 script_thread.js，并传入参数。
  • Debug \ bin \ demo1 \ _js \ script_thread.js：新线程的代码文件，通过主线程传入的参数，主线程和新线程之间可以互相调用。
 界面动态效果 点击展开
  • Debug \ bin \ demo1 \ _js \ float_title.js：实现标题的浮动效果。
  • Debug \ bin \ demo1 \ _js \ setting_blocks.js：实现内容的动态伸缩。
  • _css \ main.css：实现鼠标 hover 的动态效果。
 本演示程序 点击展开
  • LocalJSBootstrap \ LocalJSBootstrap.dsp：LocalJS 启动程序的 Visual C++ 6 项目文件。该程序适用于绝大多数情况。
  • Debug \ bin \ demo1 \ index_cn.html：演示程序的主页面。
  • Debug \ bin \ demo1 \ intro_cn.html：本页面。
  • Debug \ bin \ LocalJSBootstrap.html：LocalJSBootstrap.exe 所调用的页面，跳转到主页面。
 其他 点击展开
  • LocaJSDemo.dsw：包含 LocalJSBootstrap \ LocalJSBootstrap.dsp 和 LocalJSDemoDll \ LocalJSDemoDll.dsp 的 Visual C++ 6 工程文件。
  • compress_js.pl：调用 YUI Compressor 的示例 Perl 程序。
