LocalJS Demo Application

Run Debug\bin\LocalJSBootstrap.exe or Release\bin\LocalJSBootstrap.exe

1. JavaScript files under folder Release\bin\demo1\_js, Release\bin\demo1\localjs, and css file under folder Release\bin\demo1\_css have been compressed by YUI compressor.
2. compress_js.pl is perl script calling YUI compressor to compress js and css file, written by me, attached as reference for how to use YUI compressor.
3. The demo application saves settings under "user folder\Application Data\LocalJSDemo".
4. demo_ws.php is source file of the sample web service called by this demo application.
5. Folder LocalJSBootstrap contains source code for LocalJS bootstrap program and Visual C++ 6 project file.
6. Folder LocalJSDemoDll contains source code of sample DLL calling back and forth with JavaScript functions, along with Visual C++ 6 project file for the DLL.

You can reach me at support@localjs.org.
