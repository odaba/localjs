WPF 的華麗，Native C++ 的輕巧      只需要  HTML+CSS+JavaScript
LocalJS 演示程序文件列表
 LocalJS Library 點擊展開
  • Debug \ bin \ LocalJS.dll：LocalJS 的 DLL 文件。
  • Debug \ bin \ localJS.cfg：localJS 的配置文件，必須和 LocalJS.DLL在同一目錄下。
 LocalJS JavaScript Open Source Library 點擊展開
  • Debug \ bin \ demo1 \ localjs \ localjs_file.js：實現 namespace LOCALJS.FILE，提供了常見的文件操作接口。
  • Debug \ bin \ demo1 \ localjs \ localjs_ui.js：實現 namespace LOCALJS.UI，提供了常見的界面功能接口。
  • Debug \ bin \ demo1 \ localjs \ localjs_webservice.js：實現 namespace LOCALJS.WEB_SERVICE，提供了 web service 調用接口。
 在註冊表和 INI 文件中保存設置 點擊展開
  • Debug \ bin \ demo1 \ _js \ global_functions.js：在用戶個人目錄下創建子目錄，用於保存 INI 文件。
  • Debug \ bin \ demo1 \ _js \ program_settings.js：在註冊表中讀、寫、刪除鍵值。讀寫 INI 文件中的整數值。
  • Debug \ bin \ demo1 \ _js \ personal_settings.js：讀寫 INI 文件中的字符串。
 截取來自 COM 事件和窗口函數的退出通知，讓用戶有機會確認退出 點擊展開
  • Debug \ bin \ demo1 \ _js \ exit_confirm.js：LOCALJS.UI 把複雜的 COM 事件處理和窗口函數消息截取包裝成一個簡單的回調函數：LOCALJS.UI.exitCallback。
 創建新窗口及多個窗口之間的互操作 點擊展開
  • Debug \ bin \ demo1 \ _js \ demo_newwindow.js：在 DHTML 消息處理函數中用 LOCALJS.UI.createBrowser 函數創建新窗口，通過 initFunction 參數，將主窗口中定義的一個 JavaScript 函數注入到新窗口中執行，從而實現窗口之間的互操作。
  • Debug \ bin \ demo1 \ new_window.html：demo_newwindow.js所打開的頁面。
  • Debug \ bin \ demo1 \ _js \ demo_google.js：當用戶選擇在新窗口中打開鏈接的時候，可以通過回調函數 LOCALJS.UI.newWindowCallback 來決定是否取消這次點擊；或者創建一個新窗口，在此新窗口中打開此鏈接。
 利用 ADO 對象訪問數據庫 點擊展開
  • Debug \ bin \ demo1 \ _js \ demo_ado.js：打開數據庫演示窗口。如果數據庫演示窗口已經打開，則切換到該窗口。
  • Debug \ bin \ demo1 \ data_table.html：數據庫演示窗口的頁面文件。
  • Debug \ bin \ demo1 \ _js \ demo_datatable.js：用 ADO 對象打開數據庫並在漢化的 Yahoo YUI Library 的 DataTable 控件中排序、編輯。
  • Debug \ bin \ demo1 \ sample data.xls：作為數據庫的 Excel 文件。
  • Debug \ bin \ demo1 \ yui：Yahoo YUI Library 的一部分。
 調用任意 domain 的 web service 點擊展開
  • Debug \ bin \ demo1 \ _js \ demo_webservice.js：通過 LOCALJS.WEB_SERVICE.callUrl 調用示例 web service，利用 JavaScript 的特點非常簡單地將 JSON 格式的字符串轉變為 JavaScript 中的變量。
  • demo_ws.php：被調用的示例 web service 的源程序。
 調用 Windows API 並和用戶創建的 DLL 中的函數互相調用 點擊展開
  • Debug \ bin \ demo1 \ _js \ demo_structure.js：調用 LOCALJS.UI 中的函數移動窗口和打開文件選擇對話框。
  • Debug \ bin \ demo1 \ _js \ demo_calljs.js：調用演示程序附帶的 LocalJSDemoDll.dll 所輸出的函數 callJSFunctionInNewThread；callJSFunctionInNewThread 和 JavaScript 中的函數互相調用。
  • LocalJSDemoDll \ LocalJSDemoDll.dsp：LocalJSDemoDll 的 Visual C++ 6 項目文件。
 創建 native script 線程，主線程和新線程之間互相調用 點擊展開
  • Debug \ bin \ demo1 \ _js \ demo_thread.js：在新線程中執行 script_thread.js，並傳入參數。
  • Debug \ bin \ demo1 \ _js \ script_thread.js：新線程的代碼文件，通過主線程傳入的參數，主線程和新線程之間可以互相調用。
 界面動態效果 點擊展開
  • Debug \ bin \ demo1 \ _js \ float_title.js：實現標題的浮動效果。
  • Debug \ bin \ demo1 \ _js \ setting_blocks.js：實現內容的動態伸縮。
  • _css \ main.css：實現鼠標 hover 的動態效果。
 本演示程序 點擊展開
  • LocalJSBootstrap \ LocalJSBootstrap.dsp：LocalJS 啟動程序的 Visual C++ 6 項目文件。該程序適用於絕大多數情況。
  • Debug \ bin \ demo1 \ index.html：演示程序的主頁面。
  • Debug \ bin \ demo1 \ intro.html：本頁面。
  • Debug \ bin \ LocalJSBootstrap.html：LocalJSBootstrap.exe 所調用的頁面，直接跳轉到主頁面。
 其他 點擊展開
  • LocaJSDemo.dsw：包含 LocalJSBootstrap \ LocalJSBootstrap.dsp 和 LocalJSDemoDll \ LocalJSDemoDll.dsp 的 Visual C++ 6 工程文件。
  • compress_js.pl：调用 YUI Compressor 的示例 Perl 程序。
